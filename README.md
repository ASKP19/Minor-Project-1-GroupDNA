# 📱 GroupDNA - WhatsApp Chat Analyzer

A beginner-friendly Python and NumPy project that analyzes exported WhatsApp chats to uncover communication patterns and participant behavior.

## Features

* 📊 Message statistics
* 👥 Participant-wise analysis
* ⏱️ Response time analysis
* 🌙 Silent streak detection
* 📝 Top words analysis
* 💥 Message burst detection
* 🎭 Personality archetype detection

## Technologies Used

* Python
* NumPy
* Datetime

## How to Use

1. Export a WhatsApp chat **without media**.
2. Place the exported `.txt` file in the project folder.
3. Update the file path in the notebook if needed.
4. Run all cells in `GroupDNA.ipynb`.

## Learning Outcomes

This project was my first experience with Python and NumPy. It helped me learn file handling, dictionaries, lists, functions, datetime processing, and basic data analysis through a real-world dataset.
